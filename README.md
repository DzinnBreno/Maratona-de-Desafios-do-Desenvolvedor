# Maratona-de-Desafios-do-Desenvolvedor
SITUAÇÃO DE APRENDIZAGEM - Maratona de Desafios do Desenvolvedor
Objetivo:
Interpretar problemas e identificar os dados de entrada, processamento e saída.
Selecionar a estrutura de controle (condicional ou repetição) adequada para cada problema.
Aplicar operadores relacionais (>, <, ==, !=) e lógicos (E, OU, NAO) para criar regras de negócio.
Estruturar algoritmos complexos que combinam múltiplas lógicas.
Contexto:
Imagine que você foi contratado como Desenvolvedor(a) Júnior em uma "Fábrica de Software". Sua
função é resolver uma série de "tickets" (chamados) de diferentes clientes. Cada ticket é um pequeno
problema que precisa de um algoritmo para ser resolvido.
Seu Gerente de Projetos organizou os 40 tickets em categorias, da revisão básica até desafios
complexos que exigem que você combine todo o seu conhecimento.
